# Project Title

Bicycle Express is a website for local repair service that helps to get the right bike for customers needs or repear it.

## Based on

* HTML5, CSS, JavaScript ES5
* SASS preprocessor
* Gulp
* BEM methodology (Block, Element, Modifier)
